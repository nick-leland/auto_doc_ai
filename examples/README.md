# Examples

Small hand-authored fixtures and reference payloads used for manual checks live here.

- `fixtures/empty_title.json`: blank field template
- `fixtures/test_title.json`: representative sample payload
- `fixtures/test_title_short_addr.json`: short-address fixture
- `fixtures/test_title_long_addr.json`: long-address fixture

These files are useful for demos, debugging, and quick renderer validation without generating a full dataset.
