Annotator

Token

Click a box to edit it. Ctrl-click or Cmd-click to multi-select for merges. Use Add Box or n, then click two corners to create a token when OCR misses text. New OCR boxes start unlabeled. Keyboard shortcuts: 1-7 assign labels, Arrow Keys move boxes, Shift+Arrow Keys expand boxes, Ctrl/Cmd+Arrow Keys shrink boxes, Alt makes box nudges larger, and arrow controls follow the current rotation. s saves, m merges, [/] switches pages, q/e rotates, Delete removes the selected token, and Esc exits add-box mode. Reload OCR refreshes untouched OCR boxes but keeps edited, labeled, merged, deleted, and manually added tokens.
